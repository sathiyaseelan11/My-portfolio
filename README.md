# Space Portfolio

Minimal Vite + React 3D portfolio with Three.js, @react-three/fiber, @react-three/drei, Tailwind CSS and GSAP.

## Install

Windows / macOS / Linux

```bash
cd "d:/personal portfolio/space-portfolio"
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy
This project can be deployed to Vercel or Netlify. Drag the built `dist` folder or connect the repo. Vercel auto-detects Vite projects.
